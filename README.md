# Personal Health Assistant

This project is a **voice-enabled Personal Health Assistant** built using **Python** and **Streamlit**.
It allows users to interact through **voice input**, analyzes symptoms using datasets, and provides:

- Suggested health advice
- Recommended doctors
- Shopping list assistance
- Voice-based interaction and guidance

The system uses lightweight dataset-driven logic and a real-time Streamlit interface to offer an interactive health assistance experience.
